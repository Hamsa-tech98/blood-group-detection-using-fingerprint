# Blood Group Detection Using Fingerprint

## About the Project
This project focuses on predicting a personâ€™s blood group without collecting blood samples.
Instead, biometric fingerprint images are used along with deep learning techniques to perform
non-invasive blood group classification.

Fingerprint images are processed and analyzed using a Convolutional Neural Network (CNN),
which automatically extracts important features and classifies them into different blood group
categories.

This project was developed as an academic project to explore the application of deep learning
and image processing in healthcare-related domains.

---

## Methodology
- Fingerprint image collection (Kaggle dataset)
- Image preprocessing using Python libraries
- Feature learning using Convolutional Neural Networks (CNN)
- Model training and validation
- Blood group prediction

---

## Tools and Technologies
- Python
- Google Colab
- NumPy
- Convolutional Neural Networks (CNN)
- TensorFlow / Keras

---

## Dataset
The fingerprint dataset used in this project was sourced from Kaggle.
Due to dataset size and licensing constraints, the dataset is **not included**
in this repository.
